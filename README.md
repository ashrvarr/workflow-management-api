# Workflow Management API

A production-style backend system built with FastAPI and PostgreSQL.

## Features
- JWT Authentication
- Role-based access control
- Teams / Projects / Tasks system
- Task assignment and tracking
- Analytics endpoints (productivity metrics)
- Dockerized setup

## Tech Stack
- FastAPI
- PostgreSQL
- SQLAlchemy
- Docker

## Architecture
Team → Project → Task

## Run
docker-compose up --build
